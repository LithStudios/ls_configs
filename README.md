## Resources Without Config Files

The following resources do not have config.lua files:
- ls_dev
- ls_post_lockers
- ls_shoulder
- ls_tanks_stream
- ls_voice

## Structure

Each resource's configuration is stored in its own directory:
```
ls_configs_repo/
├── ls_bolt_minigame/
│   └── config.lua
├── ls_brake_cutting/
│   └── config.lua
├── ...
└── README.md
```

## Usage

To use these configurations, copy the desired config.lua file to its corresponding resource directory.
